

Necesitamos crear la siguiente pagina utilizando un framework de css como bootstrap para el estilo..
Estilo minimalista interactivo, tipo cintifico loco como rick y morty.
Objetivo 

Crea una página web con lógica en javascript que invierta el orden de una cadena de texto.

Ejemplo: si introduzco AI4Devs devuelve sveD4IA.

Hazlo apoyado en el seed index.html y script.js que proporcionamos dentro de la carpeta template

Ejemplo: HOLA retorna ALOH

Generar una HDU que contenga como usuario, quiero, porque, para

La HDU debe contener criterios de aceptación y criterios que no se quieren. Deja la HDU en la carpeta Template como HDU.md
En la carpeta Res contiene una imagen de ejmplo de lo que se quiere

Mis criterios deaceptaicon

Usuario escribe una palabra y se reversa en tiempo real con animación de tipado.. como si la pagina pensara.. 
La programación debe ser reactiva. si borro se quita del lado del output de la palabra..
la web debe ser mobile first
El botón debe aparecer cuando haya texto suficiente, más de 3 letras

los pasos concretos son Analizar lo que te he enviado. generar la HDU y al tenerla lista realizar el desarrollo en base a la HDU
